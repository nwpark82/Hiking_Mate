## 1. 인기 등산로 20개 콘텐츠 강화

### 📊 현황 분석
- **문제점**: 현재 663개 등산로 데이터가 있지만 상세 정보 부족
- **경쟁사 리서치 결과**: 사용자의 60%가 인터넷 카페/블로그에서 정보 획득
- **해결책**: 20개 인기 등산로에 집중하여 최고 품질의 콘텐츠 제공

### 🎯 타겟 등산로 선정 (우선순위)
- 현재 시스템의 등산로명과 일치하지 않는 부분에 대해서 데이터 검증 후 콘텐츠 강화 진행
- 인기 등산로 명은 산명+코스명으로 구성되어 있고, 현 시스템 등산로 데이터는 산명+코스번호+(정방향/역방향/왕복코스)로 구성되어 있어 타겟 등산로와의 매핑이 어려운 상황
- 현 시스템의 등산로 데이터 중 산명+코스번호가 1번만 있어 매핑이 쉬운 데이터를 우선 처리하고 매핑이 되지 않는 데이터는 추후 진행
- 등산로 리스트에서는 타겟 등산로에 대한 내용을 정리 요약하여, 한,두 문장으로 표시함
- 상세 등산로 정보에서는 콘텐츠를 강화하여 수집된 정보를 아래에서 정리한 정보 항목에 따라 표시함

### ✅ 각 등산로별 필수 정보 체크리스트

각 등산로마다 다음 정보를 **반드시** 포함:

```markdown
## 등산로 정보 템플릿

### 기본 정보
- [ ] 산 이름 (한글/영문)
- [ ] 위치 (시/군/구)
- [ ] 높이 (m)
- [ ] 난이도 (초급/중급/고급)
- [ ] 예상 소요시간 (상/중/하 체력 기준)
- [ ] 총 거리 (왕복)
- [ ] 최적 계절

### 코스 정보
- [ ] 추천 코스 3가지 이상
  - 초급자 코스
  - 중급자 코스
  - 고급자 코스
- [ ] 각 코스별 GPX 파일
- [ ] 고도 프로필 그래프
- [ ] 주요 경유지 (쉼터, 전망대, 샘터 등)

### 시각 자료
- [ ] 대표 사진 1장 (정상 또는 전망)
- [ ] 등산로 입구 사진
- [ ] 주요 명소 사진 5-10장
- [ ] 계절별 사진 (봄/여름/가을/겨울)
- [ ] 위험 구간 사진 (급경사, 암벽 등)

### 편의 시설
- [ ] 주차장 위치 및 주차 가능 대수
- [ ] 주차 요금
- [ ] 화장실 위치 (입구, 중간, 정상)
- [ ] 매점/식당 정보
- [ ] 식수대 위치
- [ ] 응급 대피소 위치

### 접근성
- [ ] 대중교통 이용 방법
  - 가장 가까운 지하철역
  - 역에서 등산로까지 버스 정보
  - 도보 이동 시간
- [ ] 자가용 이용 시 내비게이션 주소
- [ ] 주변 랜드마크

### 안전 정보
- [ ] 위험 구간 표시
- [ ] 체력 소모 구간
- [ ] 미끄러운 구간
- [ ] 낙석 위험 구간
- [ ] 일몰 시간 고려 사항
- [ ] 응급 연락처 (119, 관할 소방서)

### 추가 정보
- [ ] 베스트 포토 스팟
- [ ] 일출/일몰 명소
- [ ] 단풍 시기 (가을)
- [ ] 벚꽃/철쭉 시기 (봄)
- [ ] 겨울 설산 정보
- [ ] 주변 맛집 추천 3곳
- [ ] 등산 후 온천/찜질방

### 사용자 팁
- [ ] 초보자 주의사항
- [ ] 준비물 체크리스트
- [ ] 실제 등산 후기 (자체 작성)
- [ ] 예상 비용 (교통비, 주차비, 식비)
```

### 🎬 콘텐츠 제작 프로세스

#### Step 1: 사전 조사 (각 등산로당 2-3시간)
```
□ 기존 블로그/카페 정보 수집
□ 네이버 지도/카카오맵 리뷰 분석
□ 산림청 공식 정보 확인
□ 유튜브 영상 시청
□ 자주 묻는 질문(FAQ) 정리
```

#### Step 2: 현장 답사 (각 등산로당 4-6시간) - *추후 진행예정
```
□ 실제 등산 진행
□ 사진 촬영 (최소 30장)
  - 입구 표지판
  - 주요 경유지
  - 위험 구간
  - 전망 좋은 곳
  - 편의시설
□ GPS 트래킹 실행
□ 메모 작성
  - 소요 시간 실측
  - 체감 난이도
  - 주의사항
```

#### Step 3: 콘텐츠 작성 (각 등산로당 2-3시간)
```
□ 데이터베이스에 정보 입력
□ 상세 설명 작성 (최소 800자)
□ 사진 업로드 및 캡션 추가
□ GPX 파일 업로드
□ 내부 링크 연결 (관련 등산로, 비슷한 난이도 등)
```

#### Step 4: 검수 및 보완 (각 등산로당 1시간)
```
□ 오타/탈자 확인
□ 정보 정확성 검증
□ 사진 품질 확인
□ 모바일 뷰 확인
□ 링크 작동 확인
```

### 🔤 번역 가이드라인

#### 등산 용어 표준화
```
한국어 → 영어 (일관성 유지)

산 → Mountain (Mt.)
봉우리 → Peak / Summit
능선 → Ridge
계곡 → Valley
약수터 → Spring / Water Source
헬기장 → Helipad
대피소 → Shelter / Emergency Shelter
정상 → Summit / Peak
갈림길 → Trail Junction
쉼터 → Rest Area
전망대 → Observatory / Viewpoint

초급 → Easy / Beginner-Friendly
중급 → Moderate / Intermediate
고급 → Difficult / Advanced
급경사 → Steep Section
암벽 → Rock Face / Cliff
```

#### 거리/시간 표기
```
5.2km → 5.2km (3.2 miles)
2시간 30분 → 2.5 hours
500m → 500m (1,640 ft)
```

#### 문화적 맥락 추가
```

✅ 좋은 예: "After hiking, try Makgeolli (Korean rice wine) at a local restaurant"

❌ 나쁜 예: "김밥 싸오세요"
```

```javascript
// Intersection Observer API 활용
const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      entry.target.src = entry.target.dataset.src;
      observer.unobserve(entry.target);
    }
  });
});

document.querySelectorAll('img[data-src]').forEach(img => {
  observer.observe(img);
});
```
