# 등산 안전 수칙 완벽 가이드

등산은 건강하고 즐거운 활동이지만, 한순간의 방심이 큰 사고로 이어질 수 있습니다. 이 가이드에서는 안전한 산행을 위한 모든 정보를 단계별로 상세히 안내합니다.

## 출발 전 필수 체크리스트

### 1. 날씨 확인 (가장 중요!)

**확인해야 할 정보:**
- 기온, 강수 확률, 풍속
- 낙뢰 예보
- 미세먼지 농도
- 일출·일몰 시간

**주의해야 할 기상 상황:**
- 강수 확률 30% 이상 → 연기 권장
- 풍속 초속 10m 이상 → 위험
- 기온 영하 10도 이하 → 동상 위험
- 미세먼지 '나쁨' 이상 → 연기 권장

**날씨 확인 사이트:**
- 기상청 날씨누리
- 산악기상정보 (Korea Mountain Weather)
- 하이킹메이트 앱 (등산로별 실시간 날씨)

### 2. 등산 계획 수립

**명확한 계획 세우기:**
- 출발 시간, 하산 예정 시간
- 등산로 코스 (주 코스 + 비상 하산로)
- 중간 쉼터 위치
- 비상 연락처

**가족에게 알리기:**
- 어느 산에 가는지
- 몇 시에 출발, 몇 시까지 하산 예정인지
- 동행자는 누구인지

### 3. 체력 확인

**자신의 체력 수준 파악:**
- 초보자: 왕복 3시간 이내, 경사 완만한 코스
- 중급자: 왕복 5시간 이내, 중간 난이도
- 상급자: 6시간 이상, 높은 난이도

**컨디션 체크:**
- 감기, 몸살 → 등산 금지
- 술 마신 다음 날 → 등산 금지
- 수면 부족 → 위험
- 생리통, 근육통 → 가벼운 코스로 변경

## 산행 중 기본 수칙

### 1. 등산로 이탈 금지

**이탈하면 안 되는 이유:**
- 조난 가능성 급증
- 야생동물 조우 위험
- 낙석, 절벽 추락 위험
- 구조 어려움

**만약 길을 잃었다면:**
1. 즉시 멈춰서 침착하게 생각
2. 왔던 길로 되돌아가기
3. 계곡으로 내려가지 말 것 (급류 위험)
4. 능선을 따라 이동
5. 119 신고 (위치 최대한 상세히)

### 2. 적절한 속도 유지

**올바른 등산 속도:**
- 대화 가능한 속도 (숨이 턱까지 차면 너무 빠름)
- 본인 페이스 유지 (남과 비교 금지)
- 15-20분마다 5분 휴식

**속도 조절 실수:**
- 처음에 빨리 → 후반부 탈진
- 동행자 페이스에 무리하게 맞춤 → 사고 위험

### 3. 수분·영양 보충

**물 마시기:**
- 갈증 느끼기 전에 미리 마시기
- 15-20분마다 2-3모금
- 한꺼번에 많이 마시지 말기

**행동식 섭취:**
- 30분~1시간마다 조금씩
- 초콜릿, 견과류, 에너지바
- 점심은 정상이 아닌 8부 능선에서

### 4. 하산길이 더 위험

**통계:**
- 등산 사고의 70%는 하산 중 발생
- 무릎 부상, 미끄러짐, 낙석

**하산 시 주의사항:**
- 더 천천히 내려가기
- 무릎 각도 유지 (너무 굽히지 말기)
- 등산 스틱 적극 활용
- 돌멩이 밟지 않기 (미끄러짐)
- 낙엽 많은 길 조심 (숨은 구멍)

## 계절별 안전 수칙

### 봄 (3-5월)

**주의사항:**
- 산불 조심 (담배, 버너 사용 금지)
- 진드기 조심 (풀밭에 앉지 말기)
- 일교차 대비 보온 옷 챙기기
- 봄철 건조기 탈수 주의

**진드기 예방:**
- 긴팔, 긴바지 착용
- 밝은 색 옷 (진드기 발견 쉬움)
- 산행 후 즉시 샤워
- 물린 곳 발견 시 병원 방문

### 여름 (6-8월)

**주의사항:**
- 낙뢰 위험 (여름철 산악 사고 1위)
- 폭우 시 계곡 급류
- 벌·뱀 조우
- 열사병

**낙뢰 대피 요령:**
- 천둥 소리 들리면 즉시 하산
- 정상, 능선 피하기
- 나무 아래 피하지 말 것
- 금속 물체(스틱, 삼각대) 멀리하기
- 낮은 자세로 앉기

**벌 만났을 때:**
- 절대 손으로 쫓지 말기
- 천천히 후진
- 공격받으면 옷으로 머리 감싸기
- 쏘이면 침 제거 후 냉찜질

**뱀 만났을 때:**
- 건드리지 말고 피하기
- 물렸다면 즉시 119 신고
- 물린 부위를 심장보다 낮게
- 절대 입으로 독 빨아내지 말기

### 가을 (9-11월)

**주의사항:**
- 낙엽길 미끄러짐
- 일몰 시간 빠름
- 갑작스러운 기온 하강
- 단풍 시즌 인파

**낙엽길 안전:**
- 낙엽 밑 구멍 주의
- 보폭 좁게, 천천히
- 등산 스틱 필수

### 겨울 (12-2월)

**주의사항:**
- 동상, 저체온증
- 빙판길 미끄러짐
- 눈사태
- 조기 일몰 (오후 5시)

**저체온증 예방:**
- 땀 관리 (땀 흡수 시 급격한 체온 저하)
- 여벌 옷 챙기기
- 따뜻한 물 보온병
- 핫팩 여러 개

**빙판길 안전:**
- 아이젠 필수 착용
- 하산 시 더욱 조심
- 경사 급한 곳은 지그재그로
- 암벽 우회로 이용

## 응급 상황 대처법

### 1. 조난 상황

**즉시 해야 할 일:**
1. **119 신고**
   - "등산 중 길을 잃었습니다"
   - 산 이름, 출발 코스
   - 마지막으로 본 이정표
   - 배터리 상태

2. **위치 파악**
   - 휴대폰 GPS 확인
   - 주변 지형지물 설명
   - 하이킹메이트 앱으로 현재 위치 전송

3. **현 위치에서 대기**
   - 함부로 이동하지 말 것
   - 체력 보존
   - 휴대폰 배터리 절약

4. **구조 신호**
   - 호루라기 (3회 연속)
   - 휴대폰 플래시
   - 옷 흔들기

### 2. 부상 응급 처치

**근육 경련 (쥐):**
- 즉시 멈춰서 스트레칭
- 미네랄 보충 (이온 음료)
- 충분한 휴식

**발목 염좌:**
- RICE 요법
  - Rest: 즉시 휴식
  - Ice: 냉찜질 (차가운 물에 수건)
  - Compression: 압박 붕대
  - Elevation: 심장보다 높이

**물집:**
- 터트리지 말고 그대로
- 밴드 또는 테이프로 보호
- 터졌다면 소독 후 밴드

**찰과상:**
- 깨끗한 물로 씻기
- 소독약 바르기
- 밴드 부착

**탈수:**
- 그늘에서 휴식
- 물 천천히 마시기
- 이온 음료 섭취
- 회복 안 되면 하산

### 3. 심각한 상황

**골절:**
- 움직이지 말고 119 신고
- 부목 대기 (등산 스틱 이용)
- 체온 유지

**의식 불명:**
- 즉시 119 신고
- 기도 확보 (목 젖히기)
- 회복 자세 (옆으로 눕히기)
- 호흡 확인

## 필수 안전 장비

### 기본 장비
- 휴대폰 + 보조배터리
- 호루라기
- 헤드랜턴
- 구급약

### 응급 키트 구성
- 일회용 밴드 (5개)
- 소독약
- 거즈, 붕대
- 진통제
- 파스
- 삼각건 (골절 시)

### 비상식량
- 초콜릿
- 에너지바
- 견과류
- 사탕

## 등산 10대 안전 수칙

1. **등산 계획을 가족에게 알리고 출발한다**
2. **날씨를 확인하고, 악천후 시 등산을 미룬다**
3. **등산로를 이탈하지 않는다**
4. **해지기 전에 하산을 완료한다**
5. **적절한 장비를 착용한다**
6. **무리하지 않고 본인 체력에 맞게 산행한다**
7. **충분한 수분과 행동식을 섭취한다**
8. **산불 조심 (흡연, 화기 사용 금지)**
9. **야생 동식물을 건드리지 않는다**
10. **쓰레기는 반드시 되가져온다 (산불 예방)**
